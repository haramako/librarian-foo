# A librarian sample

A sample of 'librarian'.
https://github.com/applicationsonline/librarian

## How To Build

  $ bundle install

## How To Use

Write your Foofile (like Gemfile), and execute

  $ ./librarian-foo install

Then packages are installed './foo' directory.
